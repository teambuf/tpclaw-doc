# 智能体管理

TPCLAW 系统默认内置一个**主智能体**（TPClaw），你可以直接使用它，也可以创建新的智能体来满足不同场景需求。

## 创建智能体

点击左侧菜单「智能体」进入智能体管理页面，点击右上角「+ 新建」按钮。

![创建智能体](/img/agent/0.创建智能体.png)

在弹出的编辑对话框中配置智能体，对话框顶部有 6 个标签页：

| 标签页 | 功能 |
|--------|------|
| **基础** | 配置智能体 ID、名称、图标、角色描述 |
| **模型** | 选择大模型供应商和模型 |
| **提示词** | 编辑系统提示词 |
| **工具** | 配置智能体可使用的工具 |
| **技能** | 管理智能体的本地技能 |
| **工作空间** | 查看和编辑智能体的工作空间文件 |

## 编辑智能体

### 基础信息

![基础信息](/img/agent/1.编辑智能体-基础信息.png)

| 字段 | 说明 |
|------|------|
| **智能体 ID** | 智能体的唯一标识，创建后不可修改 |
| **智能体名称** | 显示名称，如「TPClaw」 |
| **图标** | 选择一个图标来标识智能体 |
| **角色描述** | 简要描述智能体的角色和能力 |

### 模型配置

![模型配置](/img/agent/2.编辑智能体-模型.png)

- **大模型供应商**：从下拉列表中选择已配置的供应商（如「智谱 Coding」「OpenAI」等）
- **模型**：选择该供应商下的具体模型
- **模型参数**：点击「展开模型参数」可调整温度、top_p、最大 token 等参数
- **扩展参数**：点击「展开扩展参数」可配置更多高级选项

> 供应商和模型在「设置 → 大模型配置」中管理，详见 [模型配置](/guide/configuration/models)。

### 提示词

![提示词](/img/agent/3.编辑智能体-提示词.png)

系统提示词定义了智能体的行为和角色。支持通过模板变量引用工作空间文件：

- `${include(文件路径)}` — 引入文件内容
- `${fileExists(文件路径)}` — 判断文件是否存在

默认提示词会引用工作空间中的 IDENTITY.md、AGENTS.md、SOUL.md 等文件，实现模块化管理。

### 技能

![技能](/img/agent/5.编辑智能体-技能.png)

每个智能体有独立的技能管理，分为两个作用域：

- **全局技能**：所有智能体共享的技能
- **本地技能**：仅当前智能体可用的技能

### 工作空间

![工作空间](/img/agent/1.编辑智能体-工作空间-可以修改和查看智能体的输出文件.png)

工作空间是智能体的「大脑」，包含身份、记忆、工具说明等配置文件。左侧为文件列表，右侧为编辑区域，支持：

- 查看、编辑、预览文件内容
- 添加新文件或文件夹
- 重置文件到初始状态

> 工作空间文件的详细说明请参考 [工作空间结构](/guide/workspace/structure)。
