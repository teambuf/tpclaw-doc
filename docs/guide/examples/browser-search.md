# 示例：浏览器搜索

让智能体使用浏览器工具搜索网络信息并整理结果。

## 前提：添加浏览器工具

默认智能体不会加载 `browser_use` 工具，需要手动添加：

1. 点击左侧菜单「智能体」，编辑目标智能体
2. 切换到「工具」标签页
3. 在内置工具列表中勾选 `browser_use`
4. 点击「保存」

![添加browser_use工具](/img/chat/browser_use/0.编辑智能体-工具-添加browser_use工具.png)

> `browser_use` 会启动真实浏览器进行网页操作，支持网页导航、内容提取、表单填写等。

---

## 使用示例

直接用自然语言描述搜索需求，例如：

> 帮我去搜一下2026年春晚节目单

![浏览器搜索](/img/chat/browser_use/1.浏览器搜索.png)

智能体会自动完成：

1. **启动浏览器** — 调用 `browser_use` 工具打开浏览器
2. **搜索关键词** — 在搜索引擎中输入关键词进行搜索
3. **提取内容** — 从搜索结果页面提取相关信息
4. **整理回复** — 将信息整理为结构化的表格或列表返回

## 其他可用的搜索指令

| 指令 | 说明 |
|------|------|
| `帮我搜索最近的AI新闻并总结` | 搜索新闻并生成摘要 |
| `搜索XX产品的最新价格` | 查询商品价格信息 |
| `帮我去XX网站查一下今天的天气` | 访问特定网站获取信息 |
| `帮我查一下XX技术文档` | 搜索技术文档 |

> 以上指令在网页聊天和飞书/企业微信中都可以使用，用法完全一样。
