# 自动化任务示例

通过定时任务让智能体自动执行周期性操作。

## 场景描述

- 定时检查系统状态并推送通知
- 每日自动生成报告

## 创建定时任务

直接在对话中告诉智能体你的需求，无需手写 Cron 表达式：

> 帮我设置一个定时任务，每5分钟检测电脑状态，然后通过飞书渠道发给我

![设置定时任务](/img/cron/0.设置定时任务.png)

智能体会自动完成：
1. 解析需求（频率、内容、通知方式）
2. 创建定时任务并启用
3. 返回任务 ID、频率、状态等信息

## 更多指令示例

| 场景 | 指令 |
|------|------|
| 每日早报 | "每天早上8点生成今日早报，发到飞书群 oc_xxx" |
| 工作日提醒 | "工作日早上9点提醒我开早会" |
| 系统监控 | "每5分钟检查服务器磁盘和内存，异常时发飞书通知" |
| 定时报告 | "每天下午6点生成今日工作总结发到飞书" |

> 以上指令在网页聊天和飞书/企业微信中都可以使用，用法完全一样。

## CLI 管理

也可以通过命令行管理定时任务：

```bash
# 创建
tpclaw cron create "系统监控" -s "0 */10 * * * *" -m "检查系统状态" -a main

# 查看
tpclaw cron list

# 禁用（不删除）
tpclaw cron disable <task-id>

# 删除
tpclaw cron delete <task-id>
```
