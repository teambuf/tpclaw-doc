# AGENTS.md - 工作流程

定义智能体的工作流程、协作规则和日常行为。

## 文件作用

AGENTS.md 定义：
- 日常工作流程
- 子智能体协作规则
- 任务处理方式
- 记忆和日志规则

## 模板示例

```markdown
# 工作流程

## 日常工作

### 任务处理
1. 接收用户请求
2. 分析任务类型
3. 选择合适的处理方式
4. 执行并返回结果

### 协作规则
- 复杂任务分解为子任务
- 根据任务类型调用子智能体
- 汇总结果并整合输出

## 子智能体

| 智能体 | 能力 | 调用场景 |
|--------|------|----------|
| 图片识别 | 图片理解 | 处理图片相关请求 |
| 客服 | 客户支持 | 客服相关问题 |
| 销售 | 销售支持 | 销售相关问题 |

## 记忆系统

### 长期记忆
- 重要信息存储到 MEMORY.md
- 定期整理和归档

### 每日日志
- 记录重要对话和任务
- 保存到日期命名的文件

## 心跳任务
- 每小时检查系统状态
- 定期清理过期会话
- 更新记忆索引
```

## 协作规则示例

```markdown
## 任务路由

根据用户请求自动选择处理方式：

| 关键词 | 路由目标 |
|--------|----------|
| 图片、截图、照片 | 图片识别智能体 |
| 客服、投诉、售后 | 客服智能体 |
| 销售、报价、订单 | 销售智能体 |
| 其他 | 主智能体处理 |
```
