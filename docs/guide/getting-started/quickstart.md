# 快速入门（5 分钟）

本文档将帮助您在 5 分钟内快速上手 TpClaw，体验从登录到对话的完整流程。

## 前置条件

- 已完成 [安装](./installation.md)
- 准备好 LLM 供应商的 API Key（如阿里云百炼、智谱 AI、OpenAI 等）

---

## 第一步：登录系统

启动服务后，浏览器访问 `http://localhost:9527` 进入登录页面。

![登录页面](/img/quickstart/0.登录输入密码.png)

在登录页面输入访问密码，点击「立即登录」按钮进入系统。

> 默认用户名为 `admin`，默认密码为 `admin!@#`，可在 [安全配置](/guide/configuration/security) 中修改。

---

## 第二步：配置大模型

首次登录后，系统默认已预置阿里云百炼作为大模型供应商，但需要配置 API Key 才能使用。

### 2.1 进入大模型配置

点击左侧菜单「设置」，然后选择「大模型配置」标签页。

![前往配置大模型KEY](/img/quickstart/1.前往配置大模型KEY.png)

> 如果对话时提示模型鉴权失败，也可以点击错误信息中的「前往配置」按钮直接跳转。

### 2.2 配置 API Key

系统默认已预置「阿里云百炼 Coding」供应商。找到该供应商卡片，点击编辑按钮（铅笔图标），输入您的 API Key。

![配置apiKey](/img/quickstart/2.默认是阿里云百炼供应商-配置apiKey.png)

### 2.3 添加自定义供应商（可选）

如需使用其他大模型供应商（如 OpenAI、智谱 AI 等），点击右上角「+ 添加供应商」按钮：

![添加自定义大模型供应商](/img/quickstart/3.添加自定义大模型供应商.png)

在弹出的对话框中填写：

- **供应商 ID**：从预设列表选择（如 `openai`、`zhipu`）或自定义输入
- **显示名称**：用于标识该供应商的名称
- **Base URL**：API 接口地址（选择预设供应商会自动填充）
- **API Key**：该供应商的 API 密钥
- **可用模型**：配置该供应商支持的模型列表，点击星标可设置默认模型

填写完成后点击「添加」保存。

### 2.4 设置默认供应商

点击供应商卡片上的 **星标按钮**（空心星形图标），将其设为默认供应商。设置成功后，卡片上会显示「默认」标记。

![设置默认供应商](/img/quickstart/4.点击设置成默认.png)

> 系统同一时间只能有一个默认供应商。设置新的默认供应商后，原来的默认标记会自动取消。

> 💡 **更多配置**：如需接入自定义供应商（DeepSeek、Ollama 等）或调整模型参数，请参考 [大模型配置](/guide/configuration/models)。

---

## 第三步：开始对话

### 3.1 发送消息

点击左侧菜单「对话」进入聊天页面，在底部输入框中输入消息，按 Enter 或点击发送按钮即可与智能体对话。

![聊天](/img/quickstart/5.聊天.png)

对话界面功能说明：

| 区域 | 功能 |
|------|------|
| 左侧会话列表 | 显示历史会话，支持新建、切换、删除 |
| 消息区域 | 显示对话内容，支持重新生成、复制、删除 |
| 底部输入框 | 输入消息，支持文本输入和语音输入 |

### 3.2 切换智能体

在会话列表顶部点击当前智能体名称，可展开智能体选择列表，切换到其他智能体进行对话。

![切换智能体](/img/quickstart/7.切换智能体.png)

> **多通道对话**：除了网页聊天，配置好飞书、企业微信等通道后，也可以在 IM 中直接与智能体对话，用法完全一样。详见 [飞书通道](/guide/channels/feishu)、[企业微信通道](/guide/channels/wecom)。

---

## 第四步：配置智能体

### 4.1 进入智能体管理

点击左侧菜单「智能体」进入智能体管理页面。

### 4.2 查看和编辑智能体

![智能体配置](/img/quickstart/6.智能体配置.png)

在智能体管理页面可以：

- 查看所有智能体及其配置
- 编辑智能体的名称、提示词、工具等
- 查看智能体的工作空间文件
- 管理智能体的技能

> 点击对话页面顶部的「智能体配置」按钮，也可快速跳转到当前智能体的配置页面。

---

## 第五步：接入 IM 通道（可选）

除了网页聊天，TpClaw 还支持接入飞书、企业微信等 IM 通道，让你在 IM 中直接与智能体对话，用法与网页完全一样。

### 飞书

通过长连接模式扫码绑定，无需公网 IP：

1. 在「设置」→「通道配置」中选择飞书通道
2. 使用飞书扫描弹出的二维码完成绑定
3. 绑定成功后，在飞书中直接与机器人对话

详见 [飞书通道配置](/guide/channels/feishu)。

### 企业微信

通过智能机器人 API 长连接模式接入：

1. 在企业微信管理后台创建智能机器人（选择 API 模式）
2. 获取 Bot ID 和 Secret 填入通道配置
3. 保存后在企业微信中直接与机器人对话

详见 [企业微信通道配置](/guide/channels/wecom)。

> 配置好 IM 通道后，定时任务推送也会自动发到对应的 IM 中。

## 进阶体验

### 分析项目代码

直接把项目路径发给智能体，它会自动读取目录和文件，生成项目分析报告：

> D:\github\xxx 帮我分析这个项目

详见 [示例：分析项目代码](/guide/examples/code-analysis)。

### 浏览器搜索

为智能体添加 `browser_use` 工具后，可以让它搜索网络信息：

> 帮我去搜一下2026年春晚节目单

> 默认智能体不包含 `browser_use` 工具，需要在智能体编辑器的「工具」标签页中手动勾选添加。

详见 [示例：浏览器搜索](/guide/examples/browser-search)。

### 定时任务 + 飞书推送

配置好飞书通道后，你可以用自然语言设置定时任务：

> 帮我设置一个定时任务，每5分钟检测电脑状态，然后通过飞书渠道发给我

智能体会自动创建任务、按时执行，并将结果推送到飞书。详见 [定时任务](/guide/tools/cron-task)。

> 以上示例在网页聊天和飞书/企业微信中都可以使用，用法完全一样。
