# 快速入门 (5分钟)

本文档将帮助您在 5 分钟内快速上手 TpClaw，体验核心功能。

## 前置条件

- 已完成 [安装](./installation.md)
- 准备好 LLM API Key（如 OpenAI、智谱 AI 等）

## 第一步：启动服务

### Windows

双击 `tpclaw.exe` 启动服务。

### Linux

```bash
# 启动服务
tpclaw service start

# 查看服务状态
tpclaw service status
```

启动成功后，访问 `http://localhost:9527` 进入 Web 界面。

## 第二步：登录系统

首次访问会进入登录页面：

1. 输入用户名和密码（首次使用需要注册）
2. 点击「登录」按钮

## 第三步：配置大模型

登录后，需要配置 LLM 供应商才能使用智能体功能。

### 进入设置页面

点击左侧菜单「设置」，选择「大模型配置」标签页。

### 添加供应商

1. 点击「添加供应商」按钮
2. 选择或输入供应商信息：
   - **供应商 ID**：如 `openai`、`zhipu`、`deepseek` 等
   - **显示名称**：如「智谱 AI」
   - **Base URL**：API 地址（选择预设供应商会自动填充）
3. 输入 **API Key**
4. 配置可用模型列表（可选）
5. 点击「添加」保存

### 设置默认供应商

点击供应商卡片上的星标按钮，将其设为默认供应商。

## 第四步：开始对话

### 选择智能体

1. 点击左侧菜单「对话」
2. 在智能体列表中选择一个智能体（如「主智能体」）

### 发送消息

在底部输入框中输入消息，按 Enter 或点击发送按钮。

### 功能说明

| 功能 | 说明 |
|------|------|
| 会话列表 | 左侧显示历史会话，可切换或新建 |
| 消息输入 | 支持文本、图片、文件等多种输入 |
| 快捷话题 | 预设的常用话题，快速开始对话 |
| 会话管理 | 重命名、删除、清空会话 |

## 第五步：管理智能体

### 查看智能体列表

点击左侧菜单「智能体管理」，查看所有可用智能体。

### 智能体功能

| 功能 | 说明 |
|------|------|
| 编辑配置 | 修改智能体的提示词、工具等配置 |
| 查看工作空间 | 查看智能体的记忆、技能等文件 |
| 查看技能 | 查看智能体可用的技能列表 |

### 组织架构

点击左侧菜单「组织架构」，以可视化方式查看智能体之间的协作关系。

## 第六步：管理技能

点击左侧菜单「技能管理」，可以：

- 查看所有可用技能
- 上传自定义技能
- 编辑技能内容

## 界面导航

| 菜单 | 功能 |
|------|------|
| 对话 | 与智能体进行对话 |
| 聊天室 | 多智能体协作监控 |
| 智能体管理 | 管理智能体配置 |
| 组织架构 | 查看智能体协作关系 |
| 技能管理 | 管理技能文件 |
| 设置 | 系统配置（大模型、通道、账户） |

## 移动端访问

TpClaw 支持移动端访问，自动适配手机界面：

1. 在手机浏览器中访问服务地址
2. 登录后选择智能体
3. 开始对话
