# 环境要求

TpClaw 是一个轻量级的 AI 智能体平台，对系统资源要求极低。

## 操作系统

| 系统 | 支持版本 |
|------|----------|
| Windows | Windows 10/11 |
| Linux | Ubuntu 18.04+, CentOS 7+, Debian 10+ |
| macOS | 10.15 (Catalina) 或更高版本 |

## 硬件要求

TpClaw 非常轻量，最低配置即可运行：

| 组件 | 最低要求 | 推荐配置 |
|------|----------|----------|
| CPU | 1 核 | 2 核+ |
| 内存 | 512 MB | 2 GB+ |
| 磁盘 | 200 MB | 1 GB+ |

## 软件依赖

### 二进制部署（推荐）

无需任何依赖，下载即可运行。

### 源码编译

如需从源码编译，需要：

| 软件 | 版本要求 |
|------|----------|
| Go | 1.24+ |
| Git | 任意版本 |

## 网络要求

### 端口

默认端口：`9527`

### 外部访问

需要访问 LLM API 服务（如 OpenAI、智谱 AI 等）。
